# Security Policy

## Responsible Use

VyuhScan is intended for authorized testing only.
Always obtain explicit written permission before scanning
any system you do not own. Unauthorized port scanning may
be illegal in your jurisdiction.

## Reporting a Vulnerability

If you discover a security issue in this project, please open
a private security advisory on GitHub rather than a public issue.

Go to the "Security" tab → click "Report a vulnerability"

I will respond within 48 hours.
